Departify 是一款将艺术风格应用于照片之上的应用。转换的核心是来自 Deep Art Effects 的服务。

应用的起源于 Android 新技术的练手应用，选择 DAE 作为了内容源。后因喜爱它的艺术效果，于是转而为它做了这款客户端。

我目前是 DAE 的 {departify_sub_type} 计划用户，使用他们的 API 会有限额，体现在应用中就是每台设备每天最多可以进行{departify_quota}张图片转换，而且我不能保证服务的稳定性甚至是可用性。不过我提供了使用私有密钥的接口，你可以注册成为 DAE 的开发者，根据使用情况申请你的私有密钥。点击这里可以录入你的密钥，密钥将保存于本地，只会被用于 DAE 服务的访问。 应用源码于 GitHub 公开，欢迎前往提出问题及建议。
